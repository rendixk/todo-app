# To-Do App
A modern and responsive web application designed for efficient task management. This project serves as a comprehensive tool to help users organize their daily tasks, set priorities, and track progress, all within a clean and intuitive user interface.

Installation
Follow these steps to set up and run the project on your local machine.

Prerequisites
Node.js (v14 or higher)

[Other prerequisites, if any]

Steps
Clone the repository:

```bash
git clone https://github.com/rendixk/todo-app.git
cd todo-app

Install Frontend Dependencies:

cd frontend # Or the name of your frontend folder
npm install

Install Backend Dependencies:

cd ../backend # Or the name of your backend folder
npm install

Usage
Running the Application
Start the Backend Server:

cd backend
npm run dev

Start the Frontend Client:

cd ../frontend
npm run dev
```

The application will be accessible in your web browser at http://localhost:3001.
