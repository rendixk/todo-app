To-Do App
This is a simple to-do application built with React and Vite. It allows users to manage their daily tasks, including adding, editing, and deleting to-do items.

Features
Add new tasks: Easily add new tasks to your list.

Edit tasks: Update existing tasks.

Delete tasks: Remove completed or unwanted tasks.

Mark tasks as complete: Check off tasks as you finish them.

Getting Started
Prerequisites
You'll need to have Node.js and npm or yarn installed on your machine.

Installation
Clone the repository:

Bash

git clone https://github.com/rendixk/todo-app.git
Navigate to the project directory:

Bash

cd todo-app
Install the dependencies:

Bash

npm install
or

Bash

yarn install
Running the App
To start the development server:

Bash

npm run dev
or

Bash

yarn dev
Open your browser and go to http://localhost:5173.
